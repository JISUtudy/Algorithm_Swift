# 9237 → 이장님 초대
### 문제 정리📝
1. `나무 묘목의 개수`를 입력 받는다.
2. 묘목을 `심는데` 걸리는 시간은 `1일`이 걸리며, `묘목이 다 자라는데 걸리는 시간`을 입력 받는다.
3. 입력 받은 `각각의 묘목이 자라는 시간`의 `최솟값`을 출력한다. 
4. 단, 나무 묘목을 구입한 날은 1일이다.

</br>

## 접근🚶🏻
### 나의 생각 ▾
- 시간이 오래 걸리는 모묙부터 심기위해 내림차순으로 정렬을 해준다.
- 1일부터 시작하고 묘목을 심기 위해 하루가 필요하므로, 해당 묘목이 다 자라는 날은 `1일 + 자라는데 걸리는 시간 + 심어지는 순서`이다.
- for문을 사용해 묘목의 갯수만큼 반복해주며, `max 함수`를 사용해 t에서 `가장 큰 수`를 찾아주면 된다.

</br>


### 내 코드👨🏻‍💻
```swift
let n = Int(String(readLine()!))!
var t = readLine()!.split(separator: " ").map{ Int(String($0))!}.sorted(by: >)

for i in 0..<n {
  t[i] = 1 + t[i] + (i+1)
}
print(t.max()!)
```

</br>


### Additional 📂
- Python으로 풀고 Swift로 풀었는데, Swift코드에서 max함수가 자꾸 오류가 나서 오래걸렸다. 다른 방법으로 풀기에는 시간이 부족했다. 😂 
- Swift에서 `max함수`의 반환 값은 Optional 요소이므로 `Unwrapping`해서 사용할 수 있었다.
- 어렵지 않은 문제였지만, 시간이 촉박해지니 되던 것도 안되는.. 🥲 더 열심히 해야겠다..!
